# Prompt pour créer ou améliorer le logo Shamed

**À copier-coller dans un outil de génération d'image IA** (DALL·E, Midjourney, Stable Diffusion, Adobe Firefly, etc.) ou **dans un assistant IA** pour guider la création.

---

## Prompt principal (à adapter selon l’outil)

```
Crée un logo professionnel pour "Shamed", distributeur de solutions dentaires premium au Maroc (Kerox Dental).

CONTEXTE & SECTOR :
- Entreprise B2B : distribution de matériaux dentaires (zircone, PMMA, résines 3D) pour laboratoires et prothésistes
- Positionnement : professionnel, premium, confiance, expertise médicale
- Marché : Maroc, partenariat avec Kerox Dental (leader européen)

CHARTE GRAPHIQUE À RESPECTER :
- Couleur principale (Navy) : #233C67
- Couleur accent (Cuivre/Doré) : #CD9A50
- Blanc : #FFFFFF
- Typographie : inspirée de Playfair Display (serif élégant) pour le nom
- Style : moderne, médical, premium, pas clinique ou froid

IDENTITÉ VISUELLE :
- Le nom est "Shamed" avec un point final "Shamed."
- La lettre "S" peut être mise en avant (couleur cuivre/doré) comme élément distinctif
- Éviter les symboles dentaires trop littéraux (dents, sourires caricaturaux)
- Préférer : abstraction, formes géométriques élégantes, monogramme, ou lettrage créatif

CONTRAINTES TECHNIQUES :
- Format : carré ou horizontal (ratio 1:1 ou 4:1 max)
- Fond : transparent (PNG)
- Lisible à petite taille : 42px de hauteur
- Deux versions utiles : fond clair (logo navy/cuivre) et fond sombre (logo blanc/cuivre pour nav dorée)

STYLE :
- Minimaliste, épuré
- Pas de dégradés complexes
- Lignes nettes, vectorisable
- Évoquer : excellence, précision, confiance, partenaire privilégié
```
