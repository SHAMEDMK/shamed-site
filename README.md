# Shamed – Site vitrine

Site vitrine de **Shamed**, distributeur Kerox Dental au Maroc. Solutions dentaires premium : Zircone Zircostar®, PMMA, Résines 3D, Consommables.

## Stack

- HTML5, CSS3, JavaScript
- Site statique (aucun framework)

## Structure

```
├── index.html          # Accueil
├── zircone.html
├── pmma.html
├── resine-3d.html
├── consommables.html
├── a-propos.html
├── contact.html
├── mentions-legales.html
├── politique-confidentialite.html
├── styles.css
├── script.js
├── images/
├── sitemap.xml
├── robots.txt
├── CONVENTIONS.md
└── PLAN_AMELIORATION.md
```

## Déploiement

Prévu pour déploiement sur **Vercel** (site statique).

### URL Vercel (production)

URL publique du site :

**`https://shamed-site.vercel.app`**

(Nom du projet Vercel : **shamed-site**, aligné avec le dépôt GitHub `SHAMEDMK/shamed-site`.)

Vous pouvez retirer l’ancien domaine `shamed-vtrine.vercel.app` dans Vercel → **Domains** s’il est encore listé et inutile.

## Prérequis avant production

Voir `AVANT_PRODUCTION.txt` : WhatsApp, Formspree, réseaux sociaux, pages légales.

Voir `CONVENTIONS.md` pour les règles de design, structure et ajout de contenu.
